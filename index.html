cnfsr



Cnfsr (pronounced "confessor") is a site that I worked on at VTHacks, an event at Virginia Tech in which undergraduate students in small teams created new software in 36 hours. The idea of cnfsr was inspired by so-called College Confessions websites. In such websites, students from a given college or university can post anonymous "confessions" about the college or about their life. The problem is, these sites are very unorganized; some of them are just facebook pages run by an anonymous student at the institution. Another issue is that these anonymous confessions are entrusted to some student running the site, so there isn't much privacy.

That's where cnfsr comes in. Cnfsr is intended to be a central hub for college confessions. The site is aiming to be much more centralized and organized than the individual confessions sites that currently exist. It will also protect the privaxy of its users, and not force them to submit messages via facebook (which reveals their identity) to some college student running a confessions page.

Cnfsr is currently up at cnfsr.com, but it currently has some of its main features disabled, as the backend is currently being modified. The website was originally written in HTML, CSS, Javascript, and PHP, but the PHP backend is currently being switched over to a different language. Cnfsr took home a top 5 award out of over 40 submissions at VTHacks.




TabULater






TabULater is a project that I worked on at Bitcamp, an event at the University of Maryland in which undergraduate students in small teams created new software in 36 hours.

TabULater is a Google Chrome browser extension that helps the user manage browser tabs. The user opens the extension, which loads up the tabs they already have open and presents them as a list of links. The user can then close any of their open tabs, because the links are stored in the extension and the user can reopen one of the tabs from the extension if they want. The user can also store a link to any url of their choosing by entering it in the text box. The list of tabs can be customized by closing any of the links in the extension using the "X" icons next to the links. TabULater is a fun and useful utility for Google Chrome browser users.

TabULater is now open source. The source code can be seen on my GitHub page. TabULater was written in HTML, CSS, and Javascript/jQuery. It is not currently available on the Google Play Store, because the project is still under development.

Nanbendo






Nanbendo is a project that I worked on with a friend of mine independently of any official event. With the goal of gaining knowledge of mobile development technologies and web languages (which I was unfamiliar with at the time), we started developing under the name "Nanbendo" (short for "Nathan and Ben do").

Our biggest accomplishment under this name was creating a mobile game called "Jumpy Deer", which the screenshots on the left show. Jumpy Deer is a simple game in which the user taps the screen to have the deer jump over logs and avoid birds as many times in a row as they can. Jumpy Deer was made for android and is now available on the Google Play Store. It has garnered over 100 downloads and holds a 4.5/5 star rating.

Additionally, I designed a logo and created a website for Nanbendo, nanbendo.com. The website was one of my first forays into web languages, and was coded completely from scratch in HTML and CSS. It is comprised of several different pages that describe what Nanbendo is, what our current and future projects are, and what is coming soon. While the Nanbendo project is not entirely defunct, we have decided to move on to bigger and better things. Nanbendo was a fun experiment and a great learning experience, but we are now looking to develop more professional quality apps. Still, I learned a lot from Nanbendo and Jumpy Deer, including gaining experience with Android and honing my HTML and CSS front end skills.
